# 📚 ระบบลงทะเบียนเรียน (School Registration System)
โปรเจกต์นี้คือเว็บแอปพลิเคชันสำหรับ "จัดการตารางสอน (ฝั่งครู)" และ "ลงทะเบียนเรียน (ฝั่งนักเรียน)" ออกแบบมาให้ใช้งานง่าย และสามารถรันบนคอมพิวเตอร์ของคุณเองได้เลยโดยไม่ต้องใช้อินเทอร์เน็ตครับ

## 🛠️ ขั้นตอนที่ 1: ติดตั้งโปรแกรมพื้นฐาน (ทำแค่ครั้งแรกครั้งเดียว)
ระบบนี้จำเป็นต้องใช้โปรแกรมตัวหนึ่งที่ชื่อว่า **Node.js** (เปรียบเสมือนเครื่องยนต์ที่ทำให้ระบบเว็บทำงานได้)

1. เข้าไปที่เว็บไซต์: [https://nodejs.org/en/blog/release/v16.20.2])
2. กดปุ่มดาวน์โหลดเวอร์ชัน **Windows 64-bit Installer**
3. เมื่อโหลดเสร็จ ให้ดับเบิ้ลคลิกติดตั้ง เหมือนลงโปรแกรมปกติ (กด Next ไปเรื่อยๆ จนจบ)
4. แนะนำให้ **รีสตาร์ทคอมพิวเตอร์ 1 ครั้ง** เพื่อให้ระบบรับรู้โปรแกรมใหม่


## 📥 ขั้นตอนที่ 2: ติดตั้งส่วนประกอบของเว็บ (ทำแค่ครั้งแรกครั้งเดียว)
1. เปิดเข้าไปในโฟลเดอร์ที่คุณเก็บไฟล์เว็บนี้ไว้ (โฟลเดอร์ที่มีไฟล์ `server.js`, `index.html` และ `package.json`)
2. **สำหรับ Windows:** คลิกที่แถบที่อยู่ด้านบนสุดของหน้าต่างโฟลเดอร์ (Address bar) ลบข้อความเก่าทิ้ง พิมพ์คำว่า `cmd` แล้วกด Enter
   *(หน้าต่างสีดำๆ ที่เรียกว่า Command Prompt จะเด้งขึ้นมา)*
   **(สำหรับ Mac:** เปิดโปรแกรม Terminal แล้วพิมพ์ `cd` ตามด้วยเคาะสเปซบาร์ 1 ครั้ง แล้วลากโฟลเดอร์โปรเจกต์มาวางใส่หน้าต่าง Terminal แล้วกด Enter)
3. พิมพ์คำสั่งสั้นๆ ด้านล่างนี้ลงไป แล้วกด Enter:
   npm install
รอระบบโหลดข้อมูลสักครู่ (จะเห็นว่ามีโฟลเดอร์ใหม่ชื่อ node_modules โผล่ขึ้นมา แปลว่าเสร็จเรียบร้อย!)

## 🚀 ขั้นตอนที่ 3: วิธีเปิดใช้งานหน้าเว็บ (ทำทุกครั้งที่จะใช้งาน)
1. เปิดหน้าต่างสีดำ (cmd/Terminal) ในโฟลเดอร์โปรเจกต์ตามวิธีในขั้นตอนที่ 2
2. พิมพ์คำสั่งนี้เพื่อสตาร์ทเซิร์ฟเวอร์ แล้วกด Enter:
  npm start

ถ้าระบบทำงานปกติ จะขึ้นข้อความว่า ✅ Server running at http://localhost:3000
เปิดโปรแกรมเล่นเว็บ (เช่น Google Chrome, Microsoft Edge หรือ Safari)

พิมพ์ที่ช่องใส่ URL ด้านบนสุดว่า:
http://localhost:3000 
หน้าเว็บระบบลงทะเบียนเรียนจะแสดงขึ้นมา พร้อมใช้งานทันที

## 🔐 ข้อมูลสำหรับการเข้าใช้งาน
เมื่อเข้ามาในเว็บแล้ว ระบบจะเริ่มต้นที่ "โหมดนักเรียน" เสมอ หากต้องการเข้าไปตั้งค่าระบบหรือจัดตารางสอน ให้คลิกปุ่ม "เปลี่ยนผู้ใช้" หรือ "ครู" ที่เมนูด้านซ้ายล่าง

รหัสผ่านสำหรับเข้าโหมดครู คือ: @dmin

(ข้อมูลรายวิชา รายชื่อนักเรียน และตารางสอน จะถูกบันทึกไว้ในไฟล์ school.json อัตโนมัติ หากต้องการล้างระบบใหม่ทั้งหมด ให้ลบไฟล์นี้ทิ้ง แล้วเปิดระบบใหม่ ระบบจะสร้างข้อมูลจำลองขึ้นมาให้ใหม่ครับ)

## 🛑 วิธีปิดระบบ
เมื่อเลิกใช้งานแล้ว ให้กลับไปที่หน้าต่างสีดำ (cmd/Terminal) แล้วกดปุ่มบนคีย์บอร์ด Ctrl + C จากนั้นปิดหน้าต่างสีดำทิ้งได้เลยครับ
